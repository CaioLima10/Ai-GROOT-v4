## Text comprehension protocol
1. Identify intent and context before answering.
2. Resolve ambiguous words by nearby context.
3. Detect if the user asks for definition, comparison, summary or application.
4. Respond with layered depth: short, medium, deep.

- Keep historical and doctrinal claims traceable to explicit evidence.
