Node.js é um runtime JavaScript para backend baseado no V8. Use `npm` ou `pnpm` para dependências e prefira `package-lock.json` para reprodutibilidade.

Pontos chave:
- Event loop e operações assíncronas (promises/async).
- Erros comuns: `EADDRINUSE`, versões incompatíveis, módulos ausentes.
- Boas práticas: logs estruturados, variáveis de ambiente, limites de payload.
