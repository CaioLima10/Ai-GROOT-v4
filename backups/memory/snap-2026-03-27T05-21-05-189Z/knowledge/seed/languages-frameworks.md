# Linguagens e Frameworks Prioritários

Este conteúdo define a base mínima de conhecimento técnico do GROOT.

Linguagens prioritárias:
- JavaScript/TypeScript
- Python
- Java
- C#
- Go
- Rust
- C/C++
- PHP
- Ruby

Frameworks e stacks prioritárias:
- Node.js (Express, Fastify, NestJS)
- Frontend Web (React, Vue, Angular, Svelte)
- Mobile (React Native, Flutter)
- Backend (Spring Boot, .NET, Django, FastAPI, Laravel, Rails)
- Bancos (PostgreSQL, MySQL, MongoDB, Redis)
- DevOps (Docker, Kubernetes, CI/CD, GitHub Actions)
- Cloud (AWS, GCP, Azure)

Foco de aprendizado:
- Resolver bugs e erros de build
- Instalar e atualizar dependências com segurança
- Arquitetura e boas práticas
- Desempenho e observabilidade
