CSS controla layout e visual.

Pontos chave:
- Box model: margin, border, padding, content.
- Layout moderno: flexbox e grid.
- Responsivo: media queries e unidades relativas (rem, %, vw).
- Especificidade e cascata: evite !important.
- Variaveis: :root { --cor: ... } para temas.
