HTML define a estrutura e semantica da pagina.

Pontos chave:
- Use tags semanticas: header, nav, main, section, article, footer.
- Formularios: label ligado ao input, type correto, validacao basica.
- Acessibilidade: alt em imagens, aria-* quando necessario.
- SEO basico: headings em ordem, title e meta description.
