JavaScript é linguagem base da web e também roda no backend com Node.js.

Pontos chave:
- Tipos dinâmicos e escopo por bloco (`let`/`const`).
- Async: Promises, `async/await`.
- Erros comuns: `undefined`, `null`, escopos errados e callbacks sem tratamento.
