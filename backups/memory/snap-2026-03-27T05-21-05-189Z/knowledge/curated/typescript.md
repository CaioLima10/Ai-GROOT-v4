TypeScript adiciona tipos estaticos ao JavaScript.

Pontos chave:
- type vs interface, union/intersection e generics.
- Narrowing: if/typeof/in para refinar tipos.
- Evite any; use unknown quando precisar.
- tsconfig: strict true, noImplicitAny, strictNullChecks.
