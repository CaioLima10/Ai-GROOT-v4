Docker empacota aplicações em containers.

Pontos chave:
- Use imagens oficiais e `alpine` quando possível.
- Erros comuns: portas não expostas, volumes incorretos, cache de build.
- Boas práticas: `Dockerfile` enxuto e `docker-compose` para ambientes.
