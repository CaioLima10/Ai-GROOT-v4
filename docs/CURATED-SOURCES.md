# Curadoria de Fontes Oficiais

Para adicionar conteúdos oficiais:
1. Verifique a licença e autorização.
2. Adicione o domínio em `knowledge/curated/sources.json`.
3. Crie um `.md` com front matter contendo `source`, `license` e `title`.
4. Rode `npm run knowledge:curated`.

Isso evita ingestão de material não autorizado.
