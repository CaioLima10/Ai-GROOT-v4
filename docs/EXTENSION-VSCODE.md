# Extensão VSCode (Blueprint)

Este diretório contém o esqueleto da extensão do VSCode com WebView.

## O que já está pronto
- Comando `GROOT: Abrir Chat`
- WebView com layout básico
- Configuração `groot.endpoint` e `groot.enableConnection`

## Próximos passos
- Habilitar conexão via REST ou WebSocket
- Adicionar ações: “Enviar seleção”, “Explicar erro”, “Refatorar”
- Persistir histórico local
