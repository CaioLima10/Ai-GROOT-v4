# AI Implementation Roadmap

## Fase 1 - Base profissional

- perfis da IA;
- modulos de especializacao;
- prompt builder adaptativo;
- voz mais confiavel;
- compatibilidade basica com APIs estilo OpenAI e Gemini.

## Fase 2 - Memoria e conhecimento

- ingestao dos arquivos curados no RAG;
- resumo automatico por sessao;
- aprendizagem de estilo do usuario;
- memoria por dominio.

## Fase 3 - Voz premium

- fallback local/browser;
- transcricao remota opcional;
- TTS e modo conversacional.

## Fase 4 - Produto premium

- exportacoes;
- streaming real na UI;
- perfis por workspace;
- painel de modulos e uso.

## Fase 5 - Monorepo

- separar web, api e pacotes compartilhados;
- padronizar tipos;
- CI e build por workspace.
